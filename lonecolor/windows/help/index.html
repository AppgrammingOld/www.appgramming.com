When renaming a LoneColor shortcut, the main LoneColor executable file (yes, that works too), or when using the copy/paste feature, you can use the following types of color values:

Type Shortcut name Wallpaper color
Color names LoneColor Red
Hex triplets LoneColor #FF8200
RGB triplets LoneColor 127, 0, 126
Integer values LoneColor 9110018
